## 2. 成员变量与局部变量

- 成员变量：在类内，成员方法外定义的变量
- 局部变量：在类内，成员方法内定义的变量

> 当局部变量与成员变量冲突时，优先使用局部变量

```java
package com.maqf.demo01;

public class Person {
    //成员属性（成员变量）
    public String name;
    public String gender;
    public int age;

    // 成员方法
    public void eat() {
        int age = 0; // 局部变量
        System.out.println("Eat" + age);
    }
}
```

## 3. 成员方法与静态方法（普通方法）

- 成员方法必须要实例化后通过实例.的方式调用
- 静态方法通过类.方式调用

```java
package com.maqf.demo01;  
  
public class Person {  
    //成员属性  
    public String name;  
    public String gender;  
    public int age;  
  
    // 成员方法  
    public void eat() {  
        int age = 0;  
        System.out.println("Eat" + age);  
    }  
	// 静态方法（修饰符：static）
    public static void drink() {  
        System.out.println("drink");  
    }  
}
```

```java
...
p.eat(); // 成员方法调用
Person.drink(); // 静态方法调用
...
```

## 4. 对象的赋值

```java
package com.maqf.demo03;

public class ATest {
    public static void main(String[] args) {
        A a = new A();

        // 对象是引用数据类型，b相当于是a的别名
        A b = a;
        b.a = 110;
        b.b = 220;
        
        System.out.printf("a.a: %d, a.b：%d\n", a.a, a.b);
        System.out.printf("b.a: %d, b.b：%d\n", b.a, b.b);
    }
}
```

## 7. this的用法

```java
package com.maqf.demo04;

public class Person {
    public String name;
    public int age;

    public void show() {
        // 一般来说就算成员变量没有与局部变量发生冲突也建议使用this来表明是成员变量
        System.out.printf("%s, %d", this.name, this.age);
    }

    public void setInfo(String name, int age) {
	    // 通过this使用成员变量
        this.name = name;
        this.age = age;
    }

    public void showAgain() {
        this.show(); // 方法里通过this调用成员方法
    }
}
```

```java
package com.maqf.demo04;

public class PersonTest03 {
    public static void main(String[] args) {
        Person p = new Person();
        p.setInfo("m", 19);
        p.show();
        p.showAgain();
    }
}
```

## 8. 封装

```java
package com.maqf.demo04;

public class Person02 {
    public String name;
    private int age; // private 私有成员修饰符，类外部无法使用

    // 通过公有get和set方法进行封装对私有成员属性age进行修改和访问
    public void setAge(int age) {
        if (age < 0 || age > 135) {
            System.out.println("年龄非法，将设置成默认值18！");
            this.age = 18;
            return;
        }
        this.age = age;
    }

    public int getAge() {
        return this.age;
    }
}
```

```java
package com.maqf.demo04;

public class PersonTest04 {
    public static void main(String[] args) {
        Person02 p2 = new Person02();
        p2.name = "Jack";
        p2.setAge(29);
        System.out.println(p2.getAge());
    }
}
```

## 9. 构造方法

- 与类同名
- 无返回值
- 如果不手动创建构造方法，则系统默认生成一个无参构造方法
- 构造方法不需要手动调用，在类创建对象时被自动调用
- 如果手动创建构造方法，系统则不再生成无参的构造方法
- 构造方法支持重载

```java
package com.maqf.demo05;

public class Person {
    public String name;
    public int age;

    Person() {
    }
    
	Person(String name) {
        this.name = name;
    }
    
    Person(int age) {
        this.age = age;
    }

    Person(String name, int age) {
        this.name = name;
        this.age = age;
    }

    public void showInfo() {
        System.out.printf("%s, %d\n", this.name, this.age);
    }

    @Override
    public String toString() {
        return "Person{" +
                "name='" + this.name + "\'" +
                ", " +
                "age=" + this.age +
                "}";
    }
}
```

```java
package com.maqf.demo05;  
  
public class PersonTest {  
    public static void main(String[] args) {  
        Person p = new Person("Rose", 19);  
        p.showInfo();
        // 直接打印对象时如果该对象有toString方法重写会自动执行toString方法
        System.out.println(p);   
    }  
}
```
